# React Counter

## 과제 소개

상태 관리(useState)와 조건부 렌더링을 연습할 수 있는 간단한 숫자 카운터 컴포넌트.  
숫자를 증가/감소할 수 있으며, 값에 따라 숫자 색상이 동적으로 변합니다.

---

## 주요 기능

- **숫자 증가/감소:**  
  + 버튼을 누르면 숫자가 1씩 증가, - 버튼을 누르면 1씩 감소합니다.
- **0 이하로 감소 방지:**  
  숫자는 0보다 작아지지 않습니다.
- **색상 변경:**  
  - 0일 때: 검은색  
  - 짝수일 때: 파란색  
  - 홀수일 때: 빨간색

---

## 컴포넌트 구조

- `Counter`
  - 숫자 상태 관리 (useState)
  - 증가/감소 함수
  - 색상 변경 함수 (조건부 렌더링)
  - 버튼 및 숫자 표시
